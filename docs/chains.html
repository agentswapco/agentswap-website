## Pool Grading
GRADE  DESCRIPTION
A      High liquidity, accurate quotes, heavily traded
B      Good liquidity, reliable quotes
C      Moderate liquidity, usable for smaller trades
D      Low liquidity, may have high price impact
E      Very low liquidity, probe-only (not used in direct routing)
F      Dead or unreliable, excluded from all routing

All pool types graded: V2, V3, V4, and exotic (Curve, Balancer, DODO, LB, etc.)
Ungraded pools are excluded by default. 50% price impact circuit breaker on all routes.
## Supported Tokens
AgentSwap supports any ERC-20 token with on-chain liquidity across the
indexed DEXes. Tokens are discovered automatically via factory event
scanning and graded by pool quality (A-F).

Counts (approximate):
  Ethereum  : ~2,000 tokens across 14 DEXes
  Base      : ~5,000 tokens across 29 DEXes
  Arbitrum  : ~3,000 tokens across 20 DEXes

Query the live token list:
  agentswap tokens -c base
  curl https://api.agentswap.co/api/tokens

Use token symbols or addresses in quotes:
  agentswap quote -c base -f USDC -t WETH -a 1000
  agentswap quote -c base -f 0x833589f... -t 0x420000... -a 1000
